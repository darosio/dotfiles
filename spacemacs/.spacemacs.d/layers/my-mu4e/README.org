* Description
This layer configure mu4e for my email accounts and setup following features:
  - helm-mu (for searching messages and contacts)
  - maybe persp

* Install
To use this configuration layer, add it to your =~/.spacemacs=. You will need to
add =my-mu4e= to the existing =dotspacemacs-configuration-layers= list in this
file.

* Key bindings

| Key Binding | Description       |
|-------------+-------------------|
| ~SPC m / ~  | helm-mu searching |
| ~SPC m S ~  | helm-mu searching |
| ~SPC m C ~  | helm-mu Contacts  |
